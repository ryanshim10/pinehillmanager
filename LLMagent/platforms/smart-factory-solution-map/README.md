# Smart Factory Solution Map (SK / 삼성 / LG / 포스코 / 현대차그룹)

목표: 제공된 HelloT PDF(스마트팩토리 솔루션맵)와 **동일한 톤/구성의 PPT**를 만들되, 대상은 아래 5개 그룹 기준으로 정리한다.

- SK
- 삼성
- LG
- 포스코
- 현대자동차그룹
- HD현대
- 두산그룹
- LS그룹

## 작성 원칙 (팩트/출처)

- **정확히 확인 가능한 사실만 작성**한다.
- 각 슬라이드(또는 항목)별로 **출처(URL + 문서/페이지 제목 + 확인일)**를 남긴다.
- 출처가 불명확하거나 상충하는 경우 **작성하지 않는다**(빈칸 유지).
- 추정을 해야 하는 경우:
  - 문장에 **[추정]**으로 표시
  - 추정의 근거가 되는 출처를 함께 제공
  - 근거가 없으면 추정 자체를 하지 않는다

## 생태계 맵 표현 방식

- A안: **파트너 카테고리별**(클라우드/AI/로봇/OT보안 등)
- B안: **밸류체인별**(수집→저장/파이프라인→분석/AI→실행/제어→운영)

PPT에는 A/B를 **둘 다** 넣어 비교 가능하게 구성한다.

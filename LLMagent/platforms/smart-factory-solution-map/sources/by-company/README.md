# by-company/ (그룹 내 회사별 정리)

목적: 7개 그룹(지주사) 단위를 유지하되, **그룹 내 대표 계열사(매출액 기준 TOP10)**를 선정하여 회사별로 A/B 맵, SI 및 하위 파트너십을 정리한다.

## 기준(사용자 지정)
- 기준 시점: **2024년 연매출 기준**
- 매출 기준: 기본은 **연결 매출(Consolidated)** 로 처리(별도 기준 요청 시 명시)
- 출처 우선순위: 사업보고서/분기보고서(DART), 연차보고서, IR(= L4 성과/증빙)

## 파일 구조
- `by-company/<그룹>/INDEX.md` : TOP10 리스트(매출액/출처/확인일)
- `by-company/<그룹>/<회사>.md` : 회사별 A/B 맵 + SI/파트너십 링크

## 작성 규칙
- TOP10 선정 근거(매출액)와 표는 반드시 출처(URL) + Title + Accessed(UTC)로 남김
- 회사별 문서에는 ‘출처 없는 내용 금지’ 원칙 동일 적용

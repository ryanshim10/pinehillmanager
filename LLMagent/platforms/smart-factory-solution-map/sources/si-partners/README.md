# SI 파트너십(세부) 정리 — 운영 규칙

각 그룹은 대개 내부/계열 SI 또는 주관 수행사가 존재하므로, **그 SI가 어떤 벤더/파트너와 어떤 레벨로 협력하는지**를 별도 파일로 관리한다.

## SI 업체 정의(고정)
- 업종 분류: **컴퓨터 프로그래밍 서비스업** (이 범주에 속하는 계열/외부 수행사를 SI로 간주)

## 파일 구조
- `si-partners/<그룹>.md`

## 기록 규칙
- 반드시 출처 기반(Fact + URL + Title + Accessed UTC)
- 파트너십 레벨 4단계 표기:
  - L1: MOU/발표
  - L2: 계약/파트너십
  - L3: 적용/운영(Go-live)
  - L4: 성과/증빙

## 권장 섹션
1) 그룹 내 SI 후보(계열/내부 포함)
2) SI별 세부 파트너십(클라우드/OT보안/솔루션 등)
3) To-Verify

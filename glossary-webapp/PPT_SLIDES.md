# PPT 초안(슬라이드 원고) — 제조 AI/DX 용어집 웹앱 설치/사용 매뉴얼

아래 원고는 파워포인트로 옮기면 바로 발표/교육에 쓸 수 있는 **매뉴얼 수준**의 슬라이드 구성입니다.

---

## 슬라이드 1. 제목
- 제조 AI/DX 용어집 웹앱
- 설치/운영/사용 매뉴얼
- (팀/작성일)

발표자 노트:
- 현장/사내에서 제조 AI/DX 용어를 검색하고, 엑셀로 추출/업로드/수정까지 가능한 웹앱입니다.

---

## 슬라이드 2. 목표(왜 만들었나)
- 제조 AI/DX 용어 표준화
- 현업/기획/개발 커뮤니케이션 비용 감소
- 용어 정의/예시/KPI/혼동용어를 한 곳에서 관리

---

## 슬라이드 3. 기능 요약
- 검색(키워드)
- 카테고리 필터
- 용어 상세 보기
- 용어 수정(즉시 반영)
- 엑셀 다운로드
- 엑셀 일괄 업로드
- (선택) LLM 연동: 검색 결과 없을 때 자동 생성

---

## 슬라이드 4. 구성(파일/정본)
- 웹앱 폴더: `glossary-webapp/`
- 정본 데이터: `glossary-webapp/data/glossary.json`
- 설치 직후 100개 예제 포함(바로 사용 가능)

---

## 슬라이드 5. 설치 전 준비사항
- Python 3.10+ 권장
- 브라우저(Chrome 권장)
- 사내 서버 배포 시: 8080 포트 접근 허용 필요

---

## 슬라이드 6. 설치(로컬) — Step by Step
1) 폴더 이동
   - `cd glossary-webapp`
2) 가상환경 생성/활성화
   - `python3 -m venv .venv`
   - `source .venv/bin/activate`
3) 패키지 설치
   - `pip install -U pip`
   - `pip install -r requirements.txt`
4) 실행
   - `uvicorn app.main:app --host 0.0.0.0 --port 8080`
5) 접속
   - `http://localhost:8080`

---

## 슬라이드 7. 첫 실행 확인(성공 기준)
- 웹 화면 로딩
- 검색어 입력 시 카드 리스트 출력
- 카드 클릭 시 상세 영역 출력
- 수정 버튼이 동작

---

## 슬라이드 8. 검색/필터 사용법
- 검색창: 예) OEE, MES, RAG
- 카테고리 드롭다운: 전체/전략/데이터/AI/자동화/운영/보안/성과
- 팁: “검색어 없이 카테고리만”으로도 조회 가능

---

## 슬라이드 9. 용어 수정(사용자 편집)
- 카드 클릭 → 상세 화면
- [수정] 클릭 → 편집 폼
- [저장] 클릭 → 즉시 정본(JSON) 반영

운영 팁:
- 운영 환경에서는 수정 전 백업 권장

---

## 슬라이드 10. 엑셀 다운로드
- [엑셀 다운로드] 클릭
- 현재 정본(glossary.json)을 xlsx로 내려받음

---

## 슬라이드 11. 엑셀 일괄 업로드 개요
- [파일 선택] → xlsx 선택
- [일괄 업로드(xlsx)] 클릭
- 결과: 추가/업데이트/스킵/총 용어수 표시

주의:
- 업로드는 정본을 직접 변경

---

## 슬라이드 12. 업로드 엑셀 포맷(헤더)
필수:
- 1행: 헤더
- `용어(KR)` 필수

선택 컬럼:
- `약어/EN`
- `분류`
- `한줄 정의`
- `예시`
- `KPI` (쉼표/줄바꿈)
- `혼동되는 용어` (쉼표/줄바꿈)

---

## 슬라이드 13. (선택) LLM 연동 — 언제 쓰나
- 검색 결과가 없을 때 자동으로 초안 생성
- 사내 정책에 맞게:
  - Azure OpenAI(사내 테넌트) 권장
  - 토큰은 `.env`에만 보관

---

## 슬라이드 14. (선택) Azure OpenAI 설정
- `.env.example` → `.env` 복사
- 주요 변수:
  - `LLM_MODE=azure_openai`
  - `LLM_ENDPOINT=...`
  - `LLM_API_KEY=...`
  - `LLM_DEPLOYMENT=...`

보안 주의:
- `.env`는 절대 Git에 커밋 금지

---

## 슬라이드 15. 운영/보안 권장사항
- 사내 배포 시 인증(SSO/Proxy) 권장
- 외부 통신 차단 정책 확인(LLM 사용 시)
- 정본 파일 백업/버전관리 권장

---

## 슬라이드 16. 트러블슈팅
- 설치 중 오류: venv 활성화 확인
- 접속 불가: 포트/방화벽 확인
- 업로드 실패: 헤더/필수 컬럼(용어KR) 확인
